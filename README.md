## data

JSON files that may be used during training sessions.

### astro.json

Data mainly extracted from: https://en.wikipedia.org/wiki/List_of_Solar_System_objects_by_size, using [csvtojson](https://www.npmjs.com/package/csvtojson)

#### sample

```js
{
  "name": "Jupiter",
  "radius": 69911,
  "volume": 1431280,
  "mass": 1898600,
  "density": 1.326,
  "gravity": 24.79,
  "distance": 778.3,
  "satellitesCount": 67,
  "type": "gas giant"
}
```
